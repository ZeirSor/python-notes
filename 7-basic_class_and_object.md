## 1. 类的基本语法

```python
class 类名:
    成员变量
    ...
    
    成员方法
    ...
```

### 1.1. 创建对象

```python
对象 = 类名称()
```

### 1.2. 成员方法

```python
def 方法名(self, 形参1, ..., 形参n):
    方法体
```

### 1.3. 内置方法
#### 1.3.1. `__init__`构造方法
```python
def __init__(self, ...)
```

#### 1.3.2. `__del__` 方法
- 自动调用

#### 1.3.3. `__str__`字符串方法

#### 1.3.4. `__lt__`小于/大于比较方法

```python
def __lt__(self, other):
	return self.xxx < other.xxx
```

#### 1.3.5. `__le__`小于等于/大于等于比较方法
```python
def __lt__(self, other):
	return self.xxx <= other.xxx
```
#### 1.3.6. `__eq__`等于比较方法
```python
def __lt__(self, other):
	return self.xxx == other.xxx
```
- 默认比较内存地址
- 重写比较属性

## 3. 继承

### 3.1. 单继承
```python

	...
```

### 3.2. 多继承
- 先继承优先
```python
class 类名(父类名1, 父类名2, ..., 父类名n):
	...
```

### 3.3. 复写父类成员变量和成员方法

## 4. 多态

```python

```
